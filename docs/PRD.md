# FMRO PRD (v0.1)

## Goal
Build a personal Android-first system to track robotics internship opportunities, interview progress, and deadlines with lightweight automation.

## MVP Scope

1. Job tracking
   - Company, role, city, source URL, apply URL, deadline, status
2. Interview pipeline
   - Round number, schedule time, outcome, notes
3. Personal reminders
   - Deadline reminders and interview reminders
4. Visualization
   - Kanban by stage + timeline by date
5. Crawl and review
   - Crawl selected career pages, queue results for manual approval

## Out of Scope (MVP)

- Public multi-user collaboration
- Fully automatic publish without review
- Global web-scale crawling

## Success Criteria

- Add one complete application in under 3 minutes
- Daily dashboard shows upcoming deadlines and interview tasks
- Crawler updates produce actionable review queue entries
